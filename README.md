# Rankbot

Discord bot for querying rank for VALORANT and Apex Legends player. Utilizing [mozambiquehe.re](https://apexlegendsapi.com/) and [henrikdev.xyz](https://docs.henrikdev.xyz/valorant.html) API.

1. Node.js v18 LTS
